💍 Community Matrimonial Platform

A modern community-based matrimonial web application that allows users to find suitable life partners within their community, caste, or organization. This platform supports multiple community admins, personalized profiles, and intelligent matchmaking.

📌 Features
👤 User Registration & Profile Creation
🧑‍🤝‍🧑 Create profiles for self / family members (sister, brother, daughter, etc.)
🏢 Multiple Community / Organization Support
🛡️ Admin Panel for each community
🔍 Advanced Search & Filters (religion, caste, location, etc.)
❤️ Matchmaking Algorithm
📩 Send & Accept Interest Requests
🔔 Notifications System
🔐 Secure Authentication (JWT / Firebase)
📱 PWA (Progressive Web App Support)
